git add: adicionar os arquivos ao commit.
git commit: registra os commits dos arquivos adicionados.
sem o parâmetro -m ele abre o editor de texto.
git push: envia para o repositorio remoto.
git status: mostra os estados dos arquivos do repositório
git log: mostra os histórico de arquivos alterados.
git diff: exibi a diferença entre commits e branchs
git clone: clona um repositorio remoto
git pull: atualiza o repositorio com os commits.
git checkout <numero do commit>: retorna o repositorio no commit especificado. (para sair git checkout master).
git checkout -- : reverte a alteração do arquivo antes do stage.
git checkout --HEAD : reverte as alterações depois de ser adicionado ao stage.
git checkout <nome da branch>: muda de branch
git revert: reverte um commit realizado,
git reset: retorna o repositório ao um commit anterior.(verificar as opções).
--branch (lista de commits/ ramificações do repositório)
branch master é a padrão.--
git branch : mostra as branchs do repositorio.
git branch <nome da nova branch>: cria uma nova branch.
git branch -d <nome da brach>: deleta uma branch.
git merge< nome da branch> : atualiza uma branch atual com os commits de outro merge.
